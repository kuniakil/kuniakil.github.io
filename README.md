## 🎨 設計流派 (Design Philosophy)

本站目前正進行 **Direction A: iA Writer Minimalism** 的風格實驗：
- **Typography Focused**: 使用 EB Garamond 與 JetBrains Mono 確保極致閱讀體驗。
- **Content First**: 規避 AI Slop，回歸瑞士網格系統與人文質感排版。
