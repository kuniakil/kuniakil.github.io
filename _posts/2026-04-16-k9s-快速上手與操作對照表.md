K9s 是一個終端機 UI，讓你能用鍵盤快速管理 K8s 資源。本文提供完整的使用指南。

<!-- more -->

## 第一步：進入指令模式

按下 **:** 鍵進入指令模式（切換視圖）

| 指令 | 功能 |
|------|------|
| `:ns` | 查看所有 Namespaces (行政區) |
| `:pod` | 查看所有 Pods (住戶) |
| `:deploy` | 查看所有 Deployments (部署計畫) - 開關燈在這裡! |
| `:pv` | 查看 Persistent Volumes (實體硬碟) |
| `:pvc` | 查看 Persistent Volume Claims (硬碟租約) |

## 第二步：常用快速鍵

選中對象後按下這些鍵：

| 快捷鍵 | 功能 |
|--------|------|
| `s` | **Scale (縮放)**：改數量 (0=關, 1=開) |
| `l` | **Logs (日誌)**：看程式現在正在噴什麼字 |
| `d` | **Describe**：查看詳細健康報告與錯誤原因 |
| `y` | **YAML**：直接閱讀原始設定檔內容 |
| `e` | **Edit**：直接修改設定（高手專用） |
| `Esc` | 回上一層 / 取消輸入 |
| `0` | 顯示所有 Namespace 的資源 |
| `Ctrl + d` | **Delete** 目前選中的對象 |

## 實戰情境：關掉 Nextcloud 省記憶體

1. 按下 `:` 輸入 `deploy` 並按回車
2. 用方向鍵找到 `nextcloud`
3. 按下 `s`，把數字從 1 改成 `0` 並按回車
4. 觀察 **READY** 欄位變成 `0/0`
5. 要開啟時重複一次，把 0 改回 `1` 即可

## 溫馨提醒

> 💡 K9s 是**唯讀**的嗎？不是喔！雖然它很漂亮，但它是有權力的。您在裡面按下的 `s` 或 `Ctrl+d` 是會真實影響到伺服器的。
>
> 退出 K9s 請按：**Ctrl + c**

## 安裝方式

```bash
# macOS
brew install k9s

# Linux
brew install k9s
```
