### 2. Intel Core i3-N300 / N305（中階）

| 項目 | i3-N300 | i3-N305 |
|------|---------|---------|
| 架構 | Alder Lake-N | Alder Lake-N |
| 核心/執行緒 | 8C / 8T | 8C / 8T |
| 基頻/最大頻率 | 1.8 GHz / 3.8 GHz | 1.8 GHz / 3.8 GHz |

## AMD 平台

### 1. AMD Ryzen 7020（Mendocino）— 輕量級

| 項目 | 規格 |
|------|------|
| 架構 | Zen 2 |
| 製程 | 6nm |
| 核心/執行緒 | 2C / 4T 或 4C / 4T |
| TDP | 15W cTDP |
| 內顯 | RDNA 2（最多 2 CU） |
| 記憶體 | DDR5/LPDDR5 |
| 代表型號 | Ryzen 5 7520U（4C/8T, 2.7-4.3 GHz） |

**特色**：
- 最低價的 AMD 迷你主機方案

**代表產品**：
- HP Z2 Mini G9（Chromebox）系列
- 價格區間：約 NT$3,000-5,000

## 選購要點

1. **是否需要內顯？**
   - 僅當 NAS/伺服器 → 考慮無風扇 N100
   - 需要圖形能力 → 至少選擇 AMD 6000G 以上

2. **記憶體擴充性**
   - DDR4 方案便宜，但DDR5 未來性更好
   - N100/N150 最大支援 16GB（注意是否為焊接）

3. **TDP 與散熱**
   - 6W TDP → 可被動散熱，適合 24/7
   - 45W+ → 需要主動散熱，噪音較大

4. **NVMe / SATA 擴充**
   - 確認 M.2 插槽數量與通道（PCIe 3.0/4.0）
   - 部分機種只支援一個 NVMe

5. **網路規格**
   - 家用伺服器建議選 2.5Gbps 網口
   - 需要 WiFi 6 / 6E 確認主機是否內建
