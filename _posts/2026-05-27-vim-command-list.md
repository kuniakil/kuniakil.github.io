> 💡 **提示**：Vim 有多種模式 — 普通模式、插入模式、命令列模式等。快捷鍵主要在普通模式下使用。

---

## 📂 檔案與儲存

| 快捷鍵 / 指令 | 功能說明 |
| :--- | :--- |
| `:e filename` | 開啟檔案 |
| `:w` | 儲存檔案 |
| `:q` | 退出 Vim |
| `:wq` 或 `:x` | 儲存並退出 |
| `:q!` | 不儲存強制退出 |
| `:saveas filename` | 另存為 |

---

## 🔍 移動與導航

| 快捷鍵 | 功能說明 |
| :--- | :--- |
| `h` / `j` / `k` / `l` | 左/下/上/右移動 |
| `w` | 下一個單字開頭 |
| `b` | 上一個單字開頭 |
| `0` | 移動到行首 |
| `^` | 移動到行首（非空白處） |
| `$` | 移動到行尾 |
| `gg` | 移動到檔案開頭 |
| `G` | 移動到檔案結尾 |
| `Ctrl + d` | 下移半頁 |
| `Ctrl + u` | 上移半頁 |
| `Ctrl + o` / `Ctrl + i` | 跳到上/下一個位置 |

---

## ✏️ 編輯與插入

| 快捷鍵 | 功能說明 |
| :--- | :--- |
| `i` | 在目前位置進入插入模式 |
| `a` | 在游標後進入插入模式 |
| `A` | 在行尾進入插入模式 |
| `o` | 在下方新增一行並進入插入模式 |
| `O` | 在上方新增一行並進入插入模式 |
| `r` | 取代單一字元 |
| `R` | 進入取代模式 |
| `c` | 刪除並進入插入模式 |
| `cc` | 刪除整行並進入插入模式 |
| `s` | 刪除字元並進入插入模式 |
| `S` | 刪除整行並進入插入模式 |
| `x` | 刪除目前字元 |
| `dd` | 刪除（剪下）整行 |
| `yy` | 複製（抽出）整行 |
| `p` | 在游標後貼上 |
| `P` | 在游標前貼上 |
| `u` | 復原 |
| `Ctrl + r` | 重做 |

## 🔖 標記與搜尋

| 快捷鍵 / 指令 | 功能說明 |
| :--- | :--- |
| `m{a-z}` | 設定標記（a-z） |
| `'{a-z}` | 跳到標記位置 |
| `:'{a-z}` | 跳到標記行 |
| `/pattern` | 往下搜尋 |
| `?pattern` | 往上搜尋 |
| `n` | 下一個搜尋結果 |
| `N` | 上一個搜尋結果 |
| `*` | 搜尋目前游標下的單字 |
| `#` | 反向搜尋目前游標下的單字 |

---

## 📐 區塊與文字物件

| 快捷鍵 | 功能說明 |
| :--- | :--- |
| `v` | 進入視覺模式（字元選擇） |
| `V` | 進入視覺模式（行選擇） |
| `Ctrl + v` | 進入區塊視覺模式 |
| `d` | 刪除選取範圍 |
| `y` | 複製選取範圍 |
| `c` | 更改選取範圍 |
| `aw` | 一個單字（含空白） |
| `iw` | 一個單字（不含空白） |
| `a"` | 一對引號內（含引號） |
| `i"` | 一對引號內（不含引號） |
| `a(` 或 `ab` | 一對括號內（含括號） |
| `i(` 或 `ib` | 一對括號內（不含括號） |

---

## 🪟 分頁與視窗

| 快捷鍵 / 指令 | 功能說明 |
| :--- | :--- |
| `:sp filename` | 水平分割視窗 |
| `:vsp filename` | 垂直分割視窗 |
| `Ctrl + w + s` | 水平分割 |
| `Ctrl + w + v` | 垂直分割 |
| `Ctrl + w + w` | 在視窗間切換 |
| `Ctrl + w + h/j/k/l` | 切換到左/下/上/右視窗 |
| `Ctrl + w + =` | 平均分配視窗大小 |
| `:tabnew filename` | 新增分頁 |
| `gt` / `gT` | 下一個/上一個分頁 |
| `{i}gt` | 跳到第 i 個分頁 |

## 💡 實用技巧

* `.` — 重複上一次操作
* `;` — 重複上一個 f/t 搜尋
* `,` — 反向重複 f/t 搜尋
* `Ctrl + n` / `Ctrl + p` — 自動補全（在插入模式）
* `:set number` — 顯示行號
* `:set relativenumber` — 顯示相對行號
* `/pattern\c` — 不分大小寫搜尋
