# 🔍 fzf (Fuzzy Finder) 命令列效率倍增指南

`fzf` 是一個極速、通用的**命令列模糊搜尋器（Fuzzy Finder）**。它能將任何文字清單（如：指令歷史、檔案列表、K8s Pod 列表）轉化為一個可以**打字即時篩選、預覽並選取**的互動式介面。

---

## 💡 什麼是「模糊搜尋」？
模糊搜尋代表您不需要輸入精確的完整名稱。例如您想尋找 `caddy-ingress-controller`，您只需要輸入 `cdyctl` 或 `c-i-c`，`fzf` 就會利用演算法自動幫您媒合最相似的結果。

---

## 🛠️ 三大 Zsh 核心快捷鍵（必背 🌟）

在終端機命令列中，請熟記這三個快速鍵組合，它們能為您節省大量的打字時間：

### 1. `CTRL + R` —— 歷史指令模糊搜尋
* **用法**：按下 `CTRL+R`。
* **效果**：會以互動選單列出您敲過的所有歷史指令。
* **操作**：直接打字篩選指令，用上下鍵選擇，按 **Enter** 即可將指令帶回命令列執行。
* **多選**：在列表中按 **TAB 鍵** 可以選取多個指令，按 **Enter** 會同時貼上。

### 2. `CTRL + T` —— 搜尋檔案並填入路徑
* **用法**：在指令打到一半時（例如輸入 `cat `），按下 `CTRL+T`。
* **效果**：跳出當前資料夾下的所有檔案清單。
* **操作**：輸入關鍵字找到檔案後，按 **Enter**，該檔案的相對路徑就會自動填入您的游標位置。
* **範例**：`cat ` ➡️ 按 `CTRL+T` ➡️ 搜尋 `values` ➡️ 選擇 `caddy-ingress/values.yaml` 按 Enter ➡️ 終端機自動補齊為 `cat caddy-ingress/values.yaml`。

### 3. `ALT + C` (或按 `ESC` 鍵後再按 `C` 鍵) —— 快速 cd 切換目錄
* **用法**：直接按下 `ALT+C`（在 Mac iTerm2 預設通常是按一下 `ESC` 放開，再按 `C`）。
* **效果**：跳出當前資料夾下的所有子目錄清單。
* **操作**：搜尋並選取資料夾後按 **Enter**，終端機會自動執行 `cd <該資料夾>`，瞬間完成切換。

## 🔠 搜尋語法（高階篩選過濾）

在 `fzf` 的搜尋框中，您可以使用特殊的符號來進行更精準的篩選：

| 語法範例 | 匹配類型 | 說明 |
| :--- | :--- | :--- |
| `caddy` | **模糊匹配** | 尋找含有 c、a、d、d、y 且順序相同的字串（如 `caddy`、`cfg-deploy`）。 |
| `'wordpress` | **精確匹配** | 必須完整且連續包含 `wordpress` 才會被選中（加單引號）。 |
| `^caddy` | **前綴精確匹配** | 必須以 `caddy` 開頭。 |
| `.yaml$` | **後綴精確匹配** | 必須以 `.yaml` 結尾（常用於尋找特定附檔名）。 |
| `!ingress` | **反向匹配** | 排除所有含有 `ingress` 的結果。 |

* **組合技**：多個條件可以用**空格**隔開。
  * 例如：`caddy .yaml$ !ingress` 代表：尋找含有 `caddy`、且以 `.yaml` 結尾、但排除名字裡有 `ingress` 的檔案。

## 🚀 實戰神級組合技（Pro Tips）

`fzf` 的強大之處在於它可以與任何 Linux/K8s 指令串接：

### 1. 快速查看本地任意檔案（免打完整路徑）
```bash
cat $(fzf)
```
*這會直接打開 fzf 檔案選單，選中哪一個檔案，就直接用 `cat` 印出哪一個檔案的內容。*

### 2. 互動式切換 Git 分支
```bash
git checkout $(git branch | fzf | sed "s/.* //")
```
*這會跳出所有本地 Git 分支清單，按上下鍵選中哪一個，就直接 checkout 切換到該分支。*

### 3. 互動式刪除 K8s Pod（除錯神招 ☸）
```bash
kubectl delete pod $(kubectl get pods -n gateway -o name | fzf)
```
*這會撈出 `gateway` 命名空間下的所有 Pod，讓您用關鍵字搜尋並選取，按 Enter 直接執行刪除（觸發 K8s 自動重建）。*
